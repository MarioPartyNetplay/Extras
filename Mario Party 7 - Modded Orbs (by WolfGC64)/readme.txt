Place the Mario Party 7 (USA) (v1.02) ISO / RVZ in the script's directory and run the patch script to patch it.

This patch will convert your game to one with custom orbs. Odds are REQUIRED to be set in the toolkit.

Last updated Jul 11, 2024

Custom Orb List:
- Duel: Summons a duel event
- Orb Bag: Gives you 3 random orb.
- Posion Mushroom: Slows your roll
- DK: Summons Donkey Kong
- Mystery: Gives you 1 random orb.
